Reel Forge

Topik → naskah → gambar AI → video 15–20 detik dengan efek Ken Burns dan caption bergerak. Render 100% di browser, gratis, tanpa server.

⚠️ File ini sedang dibuka langsung dari penyimpanan HP (file://). Dalam mode ini, browser akan memblokir semua permintaan ke API (naskah maupun gambar) — nanti pasti gagal dengan "Failed to fetch". Upload file ini ke hosting web (GitHub Pages, dsb) lalu buka dari alamat https://...-nya.
00 Pengaturan API ▾
Key disimpan hanya di memori browser selama sesi ini (tidak dikirim ke server manapun selain provider AI-nya langsung). Tutup tab = key hilang. Pakai tombol export di bawah untuk simpan config sebagai file, lalu import lagi kapanpun.
01 Topik & Naskah ▾
App ini tidak bisa cek fakta sendiri ke internet. Makin lengkap kamu isi kolom ini, makin kecil risiko AI mengarang nama/tanggal/angka yang salah. Untuk detail yang tidak kamu masukkan di sini, AI akan diminta bicara secara umum/tidak memastikan, bukan menebak.
Gaya ini otomatis ditambahkan ke setiap prompt gambar, plus app memakai satu "seed" acak yang sama untuk semua scene di satu video — kombinasi ini yang bikin warna & mood antar-scene senada.
